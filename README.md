# PasswordGenerator

## Prerequisites

- Elixir: 1.12.2-otp-23
- Erlang: 23.3.4
- Node: v14.13.1

## To start your Phoenix server:

- Install dependencies with `mix deps.get`
- Install Node.js dependencies with `yarn` inside the `assets` directory
- Start Phoenix endpoint with `mix phx.server`
